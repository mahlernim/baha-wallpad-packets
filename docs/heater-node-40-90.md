# 난방 노드 `40 90`

상위 문서: [README](../README.md)  
바로 가기: [프로토콜 개요](protocol-overview.md) | [하드웨어](hardware.md) | [거실 조명 `10 04`](lighting-node-10-04.md) | [현관 / 일괄소등 `1F 0F`](master-switch-node-1f0f.md) | [참고 자료](../reference/README.md)

이 노드는 BAHA 버스에서 보이는 난방 상태 / 제어 인터페이스다.

## 조회 / 응답 쌍

```text
02 40 90 01 00 D3 00 -> 02 40 90 81 ...
02 40 90 05 00 D7 00 -> 02 40 90 85 ...
```

확인된 응답 의미:

- `81` = 현재 측정 온도 테이블
- `85` = 목표 설정 온도 테이블

## 방 순서

5개 방 슬롯은 다음 순서로 확인되었다.

| 슬롯 | 방 |
| --- | --- |
| `1` | 거실 |
| `2` | 안방 |
| `3` | 놀이방 |
| `4` | 서재 |
| `5` | 침실 |

## 값 인코딩

일반 상태 값:

```text
0x80 + temperature_celsius
```

외출모드 값:

```text
0xC0 + temperature_celsius
```

예:

```text
93 = 일반 설정온도 19 C
94 = 일반 설정온도 20 C
95 = 현재 또는 설정온도 21 C
96 = 현재 또는 설정온도 22 C
97 = 현재 또는 설정온도 23 C
CA = 외출모드 설정온도 10 C
D5 = 외출모드 코드 21 C
D6 = 외출모드 코드 22 C
D7 = 외출모드 코드 23 C
```

## 응답 세부 타입

응답 세부 타입 바이트는 `DL=06` 뒤 첫 번째 바이트다.

확인된 의미:

- `02` = 모든 방이 외출모드인 상태는 아님
- `03` = 모든 방이 외출모드 상태

전부 외출모드인 경우 예:

```text
81: 02 40 90 81 06 03 D7 D6 D5 D6 D5 81 00
85: 02 40 90 85 06 03 CA CA CA CA CA 98 00
```

## 액션 / 쓰기 형식

실제로 동작하는 난방 액션 계열:

```text
02 40 90 02 06 T0 T1 T2 T3 T4 T5 CS 00
```

여기서:

- `T1..T5`는 5개 방 슬롯
- `00`으로 둔 방 바이트는 변경되지 않음

## 확인된 `T0` 의미

| `T0` | 의미 |
| --- | --- |
| `00` | 외출모드, 외출모드 해제, 일반 설정온도 쓰기에 사용하는 정상 경로 |
| `02` | 외출모드 상태인 방을 저장된 일반 설정온도로 복귀시킴. 새 설정온도 적용 경로는 아님 |

## 확인된 방별 액션

외출모드 설정:

```text
거실 외출모드   = 02 40 90 02 06 00 C0 00 00 00 00 16 00
안방 외출모드   = 02 40 90 02 06 00 00 C0 00 00 00 16 00
놀이방 외출모드 = 02 40 90 02 06 00 00 00 C0 00 00 16 00
서재 외출모드   = 02 40 90 02 06 00 00 00 00 C0 00 16 00
침실 외출모드   = 02 40 90 02 06 00 00 00 00 00 C0 16 00
```

외출모드 해제:

```text
거실 외출모드 해제   = 02 40 90 02 06 00 80 00 00 00 00 56 00
안방 외출모드 해제   = 02 40 90 02 06 00 00 80 00 00 00 56 00
놀이방 외출모드 해제 = 02 40 90 02 06 00 00 00 80 00 00 56 00
서재 외출모드 해제   = 02 40 90 02 06 00 00 00 00 80 00 56 00
침실 외출모드 해제   = 02 40 90 02 06 00 00 00 00 00 80 56 00
```

## 방별 목표 온도 쓰기

일반 목표 온도 경로는 같은 액션 계열에서 `T0=00`과 단 하나의 0이 아닌 방 바이트를 사용한다.

```text
02 40 90 02 06 00 T1 T2 T3 T4 T5 CS 00
```

예:

```text
안방 21 C = 02 40 90 02 06 00 00 95 00 00 00 43 00
```

확인된 동작:

- 일반 상태의 방에서는 저장된 목표 온도가 바뀐다.
- 외출모드 상태의 방에서는 외출모드를 해제하면서 새 목표 온도를 한 번에 적용한다.

직접 설정 가능한 범위:

```text
5 C ~ 35 C
```

## 사용자 UI에서의 "꺼짐" 해석

이 시스템에서 난방 `off` 표시는 현재 다음과 같은 의미로 보인다.

```text
외출모드 / 동파방지 10 C
```

즉, 별도의 완전 정지 상태로 보이지는 않는다.
