# 참고 자료

상위 문서: [README](../README.md)  
바로 가기: [프로토콜 개요](../docs/protocol-overview.md) | [하드웨어](../docs/hardware.md) | [거실 조명 `10 04`](../docs/lighting-node-10-04.md) | [현관 / 일괄소등 `1F 0F`](../docs/master-switch-node-1f0f.md) | [난방 `40 90`](../docs/heater-node-40-90.md)

이 디렉터리는 공개용으로 다듬기 전의 장문 조사 자료를 보존한다. 원문 로그는 조사 당시 기록을 최대한 그대로 남긴 것이어서 표현이 정제되지 않은 부분이 있다.

파일:

- [raw-session-notes.md](raw-session-notes.md): 전체 조사 로그, 테스트 이력, 패킷 해석 과정
- [heater-packets.html](heater-packets.html): 분석 과정에서 만든 난방 HTML 참고 문서

권장 사용 순서:

- 먼저 `docs/` 아래의 정리된 문서를 본다.
- 중간 추론 과정이나 더 긴 예제가 필요할 때만 여기의 자료를 참고한다.
